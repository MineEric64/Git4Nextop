## Introduction
넥스탑 코딩정보보안 학원에서 쓰이는 `GitHub Desktop`보다 안전성, 보안성, 편리성이 더 높은 깃 저장소 관리 프로그램.

또한 README를 작성할 때 극도의 노가다 작업이 필요할 경우 Git4Nextop 기능 중 Boj4Nextop 기능을 사용 시
자동으로 극도의 노가다 작업을 한 방에 처리할 수 있습니다.
